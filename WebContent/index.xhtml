Projeto JSF

JSF (JavaServer Faces) é uma especificação Java para a construção de interfaces de usuário baseadas em componentes para aplicações web.
Esta tecnologia possui um modelo de programação dirigido a eventos e permite que o programador se concentre apenas na lógica da aplicação.

Nesta aplicação temos um sistema de Cadastro de Livros com tecnologias PrimeFaces e Ajax, além da integração com a especificação CDI.
